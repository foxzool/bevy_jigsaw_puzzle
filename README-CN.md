# 拼图游戏

## 快捷键

- <kbd>PageUp</kbd> / <kbd>PageDown</kbd> 视角放大/缩小
- <kbd>Space</kbd> 显示原图提示
- <kbd>H</kbd> 显示两个匹配的拼图
